1. Compilation:

তোমার .go ফাইল কে compiler পড়ে।

Source code কে machine code (CPU-readable instructions) এ convert করে।

.exe বা binary ফাইল তৈরি হয়।

2. Execution:

Operating system ওই executable কে memory (RAM) এ load করে।

CPU সেই binary instructions step-by-step execute করে।

Source code দরকার নেই, CPU শুধু machine code execute করে।
